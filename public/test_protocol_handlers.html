Como usar os protocolos personalizados

Depois que a aplicação for instalada como PWA, você poderá usar os seguintes protocolos:

  • web+webpush://subscribe - Para ações relacionadas ao web push
  • web+notify://Hello%20World - Para enviar notificações rápidas
  • web+pushnotification://{"title":"Test","body":"Message"} - Para notificações com dados estruturados

Nota: Os protocolos personalizados só funcionam quando a aplicação está instalada como PWA.
